# 2026 Industry-Analysis Terminology

Vocabulary the skill descriptions, SKILL.md bodies, and output briefs should use. Reflects current senior MBB practice (May 2026), not 2005-era language.

| Avoid (2005) | Use (2026) | Why |
|-------------|-----------|-----|
| Industry analysis | Industry-structure analysis | Clarifies the scope: structure, not just description |
| Market sizing | Granular market sizing / G3 decomposition | Aggregate sizing without de-averaging is misleading |
| Competitive analysis | Competitive arena mapping | "Arena" is McKinsey's current term (2024 relaunch) |
| Barriers to entry | Entry/mobility barriers | Entry vs movement-between-groups are different |
| Competitive advantage | Strategic power / structural moat | "Advantage" is operational; "power" requires benefit + barrier |
| Value chain | Value chain + profit pool (always paired) | Mapping value without profit overlay is incomplete |
| Where to compete | Where to play / How to win | Lafley/Martin cascade is the current synthesis language |
| Portfolio analysis | Horizon allocation | Three Horizons replaced BCG matrix as the portfolio frame |
| Industry attractiveness | Arena qualification | "Is this an arena (high growth + dynamism) or mature?" |
| Platform companies | Ecosystem orchestrators | Distinguishes orchestrator from participant |
| Network effects | Network economies (as a Power) | Helmer's 7 Powers terminology |
| Moat | Strategic power (specify which of 7 Powers) | Generic "moat" is imprecise |
| Disruption | Discontinuity / S-curve jump | Christensen's "disruption" has been narrowed in current usage |
| Strategic planning | Strategy activation / WTP-HTW | "Planning" is dated; activation is current |
| SWOT | (avoid — use Five Forces + value chain + arenas) | SWOT is treated as pre-strategic at senior level |
| BCG Growth-Share Matrix | Three Horizons portfolio | The 2x2 was absorbed into Three Horizons logic |

## Modern frameworks to name explicitly

When the analysis uses any of these, name the framework:

- **G3 / granular growth** (McKinsey) — for market sizing
- **Strategy Palette** (BCG / Reeves) — for environment diagnosis
- **Arenas of Competition** (McKinsey 2024) — for active-market overlay
- **Profit pools** (Bain / Gadiesh-Gilbert) — for value-capture mapping
- **Three Horizons** (McKinsey) — for trajectory + portfolio
- **7 Powers** (Helmer) — for moat / durability assessment
- **Where to Play / How to Win** (Lafley-Martin) — for synthesis
- **JTBD** (Christensen / Ulwick) — for demand-side analysis
- **Power Progression** (Helmer) — for trajectory-stage moat assessment

## Frameworks that stayed (still in use, not "old")

These are 1990s-2000s frameworks still in current senior practice:
- Porter's Five Forces (+ complementors + AI-as-force extensions)
- Porter's Value Chain
- Strategic Group Mapping
- Industry Lifecycle / S-Curve
- PESTEL (as macro scan, not standalone analysis)

## AI-era addition

Treat **AI** as a named force in every Five Forces analysis (as either a sub-force inside substitutes/new-entrants or as a sixth force depending on industry). Specifically assess:
- Cost-structure impact (which activities does AI automate? what % of cost base?)
- New-entry vector (AI-native competitor with no legacy cost structure)
- Data-intermediary position (who owns the model-training data for this industry?)
